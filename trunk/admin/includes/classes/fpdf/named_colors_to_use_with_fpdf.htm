Named colors to use with the "fpdf" class

Instead of calling the color-setting-functions of "fpdf"

function SetDrawColor($r,$g=-1,$b=-1)
function SetFillColor($r,$g=-1,$b=-1)
function SetTextColor($r,$g=-1,$b=-1)
 

with the R, G and B values as parameters, they now can all be called with a color name as parameter.

This is much easier to use, as

"SetTextColor('brown')"

is much better readable then

"SetTextColor(155,42,42)"

The following color names can be used:
(Any other name will be converted to black)
